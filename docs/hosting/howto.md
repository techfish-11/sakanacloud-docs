# 利用開始方法
このガイドでは、SakanaCloud Lab Hostingサービスの利用を開始する手順について説明します。

## Discordサーバーへの参加
まず、SakanaCloud Labの公式Discordサーバーに参加してください。
利用開始にはDiscordサーバーからの申請が必要です。

https://discord.gg/pTK7fvuZC2

## 利用開始申請
Discordサーバー内の #サービス利用申請 チャンネルで、利用開始の申請を行ってください。申請には以下の情報が必要です：
- サーバーの名前（好きな名前を付けてください）
- 希望するサービスの種類（例：ウェブホスティング、Minecraftサーバーなど）
- 利用して行いたいことについての企画書
- その他必要な情報

運営が利用申請を受理したら仮利用者ロールを付与します。
#利用規約 チャンネルで利用規約を確認し、同意してください。同意後、正式な利用者ロールが付与されます。

その後、メールアドレスの登録を行い、ホスティングパネルへのアクセス情報が提供されます。

## ホスティングパネルへのアクセス
ホスティングパネルのURLとログイン情報が提供されたら、パネルにアクセスしてログインしてください。
パネル内でサーバーの設定や管理を行うことができます。
