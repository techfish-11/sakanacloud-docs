# ホスティングパネルの使い方
当組織では、管理パネルにPterodactylPanelを採用しています。
このガイドでは、PterodactylPanelを使用してサーバーを管理する方法について説明します。

## ログイン
ホスティングパネルのURLとログイン情報が提供されたら、パネルにアクセスしてログインしてください。

<img src="/img/panel-login.png" alt="Pterodactyl Login Screen"/>
USERNAME or EMAIL: ユーザー名またはメールアドレスを入力します。
PASSWORD: 運営によって作成されたパスワードを入力します。

ログイン後、/accountページからパスワードの変更を行うことをお勧めします。

## ダッシュボードの概要
ログイン後、サーバー一覧が表示されます。ここでは、利用可能なサーバーのステータスやリソース使用状況を確認できます。
<img src="/img/panel-dashboard.png" alt="Pterodactyl Dashboard"/>

## サーバーの管理
サーバーをクリックすると、サーバーの詳細ページに移動します。ここでは、以下の操作が可能です：
- **スタート/ストップ/リスタート**: サーバーの起動、停止、再起動を行います。
- **コンソールアクセス**: サーバーのコンソールにアクセスして、直接コマンドを入力できます。
- **ファイルマネージャー**: サーバーのファイルをアップロード、ダウンロード、編集できます。
- **バックアップ**: サーバーのバックアップを作成および管理します。

細かい使用方法は、お手数ですがご自身で検索して調べてください。
